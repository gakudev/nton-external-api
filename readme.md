# nton external api

rest api from iptv

## Don't Forget

- npm i = for install node_modules
- npm run dev = nodemon
- npm run start = start app

## .env_example

- REST_API=REST-API-IPTV
- APP_PORT=INTERGER
- APP_NAME=STRING
- HOST_LISTEN=0.0.0.0
- FILE_BASE_PATH=PATH-FOLDER
- LOG_PATH=PATH-FOLDER + /error.log
- LOG_LEVEL=error/fatal/info (default: error)
